# EITHER
EITHER evaluates the tumor heterogeneity level of each tumor sample based on shannon entropy


## Examples

```  
exp=read.csv("exp.csv",header=F)
match=read.csv("match.csv")

library(EITHER)
EITHER(exp, match)
```
